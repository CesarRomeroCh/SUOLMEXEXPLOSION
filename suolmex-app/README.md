
# Calculadora de Pedido SUOLMEX

Aplicación en Streamlit para generar pedidos, calcular insumos y exportar resultados a Excel.

## ¿Cómo usar?

1. Subir el archivo Excel del pedido o capturar manualmente modelo, talla y cantidad.
2. Verificar el resumen del pedido.
3. Exportar a Excel al finalizar.

---

**Tecnologías usadas:** Streamlit, Pandas, Openpyxl
